Objetivo

Este Centro de Computo tiene como proposito el poder llevar acabo varios servicios de diferentes áreas de una organización ya sea dentro de la misma o fuera de ella tales como: producción,control de operaciones, captura de datos, programación, dibujo, busqueda de información, etc.
Este espacio cuenta con el mejor equipo para tener un buen rendimiento a la hora de realizar las tareas que el usuario necesite.
